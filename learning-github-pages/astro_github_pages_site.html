Astro is a modern static site generator built for performance and flexibility. Here's how to get started:

  • Install Astro: npm create astro@latest
  • Develop your site locally: npm run dev
  • Build your static site: npm run build
  • Deploy with GitHub Actions using withastro/action.

Benefits: partial hydration, supports React/Svelte, fast performance, clean output.

Jekyll is supported natively by GitHub Pages. To use it:

  • Push to GitHub; Pages will build automatically

  • Use GitHub Actions to automate builds
