Step-by-Step Guide to Power Query

Scenario: You are a business analyst with monthly sales data stored in separate Excel files. Your goal is to combine them into one master table for a report.

Phase 1: Connect to Data

Step 1: Get Data from Folder

Instead of connecting to each file individually, connect to the entire folder.

In Excel or Power BI, navigate to Data > Get Data > From File > From Folder. This tells Power Query to treat all files in that folder as a single source.

Phase 2: Transform and Clean

Step 2: Combine Files

Power Query will show you a list of the files. Click the Combine & Transform Data button. Power Query will automatically analyze the files and suggest how to combine them based on sheet names (e.g., "Sheet1").

Step 3: Use the Query Editor

The Power Query Editor window will open. This is your visual workbench. On the right, you'll see a list of Applied Steps. Every change you make is recorded here.

  • Remove unwanted columns: Right-click the header of a column you don't need and select "Remove".
  • Correct data types: Click the icon in the column header (e.g., ABC, 123) to change text to numbers, dates, etc. This is crucial for calculations.

Phase 3: Load the Result

Step 4: Close & Load

Once your data looks clean, click the Close & Load button in the top-left. Power Query will execute all your applied steps and load the final, combined table into an Excel sheet or a Power BI data model, ready for analysis.
