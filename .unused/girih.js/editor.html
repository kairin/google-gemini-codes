Getting started

Tiling mode

Select between the five Girih tiles and start tiling. Tiling mode defaults to a plain display style to simplify tile recognition.

Styling mode

Choose between different color presets or customize the display style.

Svg export

Export your pattern as vector images for further processing and resolution independend printing.

File dialog

Save and load your patterns to your local browser storage.

Use these keyboard shortcuts for efficient tiling:

Q, W, Spacebar — Rotate mouse tile
1-5 — Select tile
Backspace — Delete last tile
E — Eraser tool
-,+ — Zoom
Arrow keys — Scroll canvas
Tab — Toggle palette
